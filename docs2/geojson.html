GeoJSON

Introduction

GeoJSON is a format for encoding a variety of geographic data structures. See geojson.org for more information.

GeoJSON supports the following geometry types: Point, LineString, Polygon, MultiPoint, MultiLineString, and MultiPolygon. Geometric objects with additional properties are Feature objects. Sets of features are contained by FeatureCollection objects.

For further information on adding different types, refer to the Leaflet GeoJSON examples.

Indoor support

Features can be set to indoors by specifying the 'indoorMapId' and 'indoorMapFloorId' properties:

{
    "geometry": {
        "type": "Point",
        "coordinates": [ -2.988967373895398, 56.45511052483176 ]
    },
    "type": "Feature",
    "id": 0,
    "properties": {
        "date": "2018-07-06",
        "indoorMapId": "westport_house",
        "indoorMapFloorId": 2
    }
}

Refer to the indoor examples for more information on displaying time-series data indoors.

Heatmap intensity

You may wish to apply intensities to specific data points, for example if you are visualising the population of cities, each city would have an associated intensity (in this case the number of inhabitants).

This can be set in the GeoJSON feature, under the intensity property. In the below example, we set the intensity of this feature to 31:

{
    "geometry": {
        "type": "Point",
        "coordinates": [ -2.988967373895398, 56.45511052483176 ]
    },
    "type": "Feature",
    "id": 0,
    "properties": {
        "date": "2018-07-06",
        "intensity": 31
    }
}
