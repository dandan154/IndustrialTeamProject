What can you do?

Time visualisation library

The aim of this piece of software is to allow you to import any data you wish which contains longitude latitude and time values, for it to then be represented as heatpoints for the data creating a heatmap. For further look into how to achieve this check the out the documentation dropdown box at the top of the screen.
