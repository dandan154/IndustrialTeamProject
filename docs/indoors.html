How to create with the software

wrld.time.js is a plugin for Wrld3d which allows for the visualisation of time series data.

To begin using the plugin, you will need to either have your data in a GeoJSON format, or convert it to one. Learn more about the GeoJSON format by visiting our GeoJSON guide.

First, setup your Wrld map:

<div id="map"></div>
<script src="https://cdn-webgl.wrld3d.com/wrldjs/dist/latest/wrld.js"></script>
<script src="wrld.time.js"></script>
<script>
    var map = L.Wrld.map("map", "[YOUR WRLD API KEY]", { 
        center: [ 56.462, -2.9707 ], 
        indoorsEnabled: true 
    });
</script>

Next, you can create a wrldTime object by passing in the Wrld map instance and your data:

wrldTime(map, data);

Data can either be the GeoJSON object, or a URL pointing to the GeoJSON file.
