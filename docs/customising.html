Customising

Time slider

Position

The position of the time slider can be changed from the wrldTime options.

To set the horizontal position, use the 'horizontalPosition' property (accepts 'left', 'center' or 'right'), and to set the vertical position use the 'verticalPosition' property (accepts 'top' and 'bottom'):

wrldTime(map, data, {
    horizontalPosition: 'left' || 'center' || 'right',
    verticalPosition: 'top' || 'bottom'
});

Native styling

CSS can be used to style the time slider component. The container is has the class '.time-slider-container', and the slider itself has the class '.time-slider'. Use '.time-slider::-webkit-slider-thumb' and '.time-slider::-moz-range-thumb' to style the thumb of the slider.

Heatmap customisation

Heatmap options can be set by setting the heatmapOptions option on the wrldTime constructor:

wrldTime(map, data, {
    heatmapOptions: {
        minOpacity: 0.5,
        maxZoom: 16,
        max: 1,
        radius: 25,
        blur: 15,
        gradient: {0.4: 'blue', 0.65: 'lime', 1: 'red'}
    }
});

The available heatmap options are:

minOpacity - the minimum opacity the heat will start at

maxZoom - zoom level where the points reach maximum intensity (as intensity scales with zoom), equals maxZoom of the map by default

max - maximum point intensity, 1.0 by default

radius - radius of each "point" of the heatmap, 25 by default

blur - amount of blur, 15 by default

gradient - color gradient config, e.g. {0.4: 'blue', 0.65: 'lime', 1: 'red'}
