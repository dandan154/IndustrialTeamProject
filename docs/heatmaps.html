Heatmaps

How to create with the software

A heatmap can be added on both indoor and outdoor maps. To create a heatmap, first initialize the Wrld map:

<div id="map"></div>
<script src="https://cdn-webgl.wrld3d.com/wrldjs/dist/latest/wrld.js"></script>
<script src="wrld.time.js"></script>
<script>
    var map = L.Wrld.map("map", "[YOUR WRLD API KEY]", { 
        center: [ 56.462, -2.9707 ], 
        indoorsEnabled: true 
    });
</script>

Once the Wrld map has been created, initialise wrld.time.js with the heatmap option set to true:

wrldTime(map, data, {
    heatmap: true
});

Intensity

Global intensity factor

Heatmaps may be used in a variety of situations. For example, if you have a sparse data set, you may wish that each point in your data set is displayed more intensely. Another example is that you should have different intensities when displaying a heatmap of cities compared to displaying a heatmap of countries.

You can also set an indoor intensity factor which is applied to indoor data points. By default the global intensity is set to 50 and the indoor heatmap intensity is set to 1. This can be overriden when constructing the wrldTime object:

wrldTime(map, data, {
    heatmap: true,
    heatmapIntensity: 80,
    indoorHeatmapIntensity: 0.6
});

Intensities on data points

You can also apply intensities to specific data points, for example if you are visualising the population of cities, each city would have an associated intensity (in this case the number of inhabitants).

This can be set in the GeoJSON feature, under the intensity property. In the below example, we set the intensity of this feature to 31:

{
    "geometry": {
        "type": "Point",
        "coordinates": [ -2.988967373895398, 56.45511052483176 ]
    },
    "type": "Feature",
    "id": 0,
    "properties": {
        "date": "2018-07-06",
        "intensity": 31
    }
}
